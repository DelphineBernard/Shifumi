Shifumi

Règles du jeu

Vous jouez contre l'ordinateur.

La partie se déroule en 3 manches.

Vous avez le choix entre :

feuille, pierre ou ciseaux.

Un point est donné à celui qui a choisi l'élément le plus fort, sachant que :

  • puits > pierre

    (la pierre tombe dans le puits)

  • puits > ciseaux

    (les ciseaux tombent dans le puits)

  • feuille > puits

    (la feuille recouvre le puits)

  • feuille > pierre

    (la feuille enveloppe la pierre)

  • pierre > ciseaux

    (la pierre casse les ciseaux)

  • ciseaux > feuille

    (les ciseaux coupent la feuille)

Si l'ordinateur et le joueur ont choisi le même élément, aucun d'eux n'emporte de point.
